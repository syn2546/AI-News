## ⭐ Trending Repos

### garrytan/gstack 📈 steady growth

Stats: ⭐ ~117k · 🍴 ~17.4k · 📈 +674 today, cooling from viral peak · 🟢 active, 49 contributors
Traction: One of 2026's fastest GitHub ramps (10k stars in 48h in March), now flattening. Real ecosystem spin-offs and many tutorials, but no marquee corporate adopters. Red flag: ~445 open PRs / ~297 issues backlog.
References:
- https://github.com/garrytan/gstack
- https://news.ycombinator.com/item?id=47355173

### hugohe3/ppt-master 🔥 exploding
An AI skill that turns any document (PDF, DOCX, URL, Markdown) into a natively editable PowerPoint — real shapes, charts, OOXML animations, plus voice-narrated speaker notes — inside Claude Code, Cursor, VS Code and Codex.
Stats: ⭐ ~33.1k · 🍴 ~2.8k · 📈 rapid, ~+600 in days · 🟢 v2.11.0 Jun 20, daily commits
Traction: Standout is maintenance velocity — last commit Jun 27, a new minor every 5–7 days, near-clear backlog (~5 issues, 1 PR). Differentiator is genuinely editable OOXML output with synced MP4 narration. Caveat: largely single-maintainer; traction is stars + skill-catalog listings, not named adopters.
References:
- https://github.com/hugohe3/ppt-master
- https://github.com/hugohe3/ppt-master/releases

### safishamsi/graphify 🔥 exploding
A coding-agent skill that turns any folder — code, SQL schemas, docs, PDFs, images, video — into a queryable knowledge graph. One `/graphify .` command emits an HTML graph, a markdown report and JSON; plugs into Claude Code, Cursor, Codex, Gemini CLI and ~20 agents.
Stats: ⭐ ~73k · 🍴 ~7.3k · 📈 exploding, ~12k in first 48h · 🟢 v0.8.50 Jun 27, 144 releases
Traction: A viral spike catalyzed by an Andrej Karpathy "LLM knowledge base" post 48h before launch, plus a Y Combinator S26 affiliation. Red flags: days-to-weeks old, single-maintainer, ~219 open PRs / ~171 issues, a public prompt-injection concern over the installed skill.md, and PyPI naming confusion (`graphifyy` vs the `graphify` command).
References:
- https://github.com/safishamsi/graphify
- https://trendshift.io/repositories/25296

### anthropics/skills 🛠️ mature & active
Anthropic's official reference repo for Agent Skills — folders of instructions, scripts and resources (SKILL.md format) that Claude loads dynamically to specialize. Doubles as a Claude Code plugin marketplace and the canonical example set for the open Agent Skills standard.
Stats: ⭐ ~156k · 🍴 ~18.4k · 📈 steady, ~+15k in a month · 🟢 last commit Jun 27, weekly upkeep
Traction: Past its Oct 2025 viral launch but still accumulating, with a high fork ratio signaling people template skills rather than just bookmark. First-party Office skills, an open portable standard (agentskills.io), and a real "awesome-claude-skills" ecosystem. Red flag: ~714 open PRs / ~275 issues — tight first-party curation, community PRs pile up.
References:
- https://github.com/anthropics/skills
- https://code.claude.com/docs/en/skills

### browser-use/video-use 🧪 early/experimental
An open-source Claude Code skill for video editing — drop raw footage in a folder, chat with the agent, get a finished `final.mp4`. The LLM never watches the video; it reads a word-level transcript and cuts on word boundaries (filler removal, color grade, fades, subtitles, overlays).
Stats: ⭐ ~10.5k · 🍴 ~1.5k · 📈 slowing, spike was Apr launch · 🟢 last commit May 15, ~6 weeks stale
Traction: Rode the browser-use brand and an "agent edits your video" hook to ~10k stars in two months, with multiple independent how-tos. But the repo has gone quiet: last commit May 15, ~33 open PRs vs 10 issues, no published releases. Idea is technically credible; adoption is hobbyist and durability unproven.
References:
- https://github.com/browser-use/video-use
- https://github.com/browser-use/video-use/blob/main/README.md
