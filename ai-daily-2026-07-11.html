›
Published 2026-07-08

Cybersecurity firm Wiz disclosed 'GhostApproval,' a vulnerability affecting AI code assistants including Cursor, Amazon Q Developer, and Google Antigravity. By exploiting Unix symbolic links (symlinks) and UI misrepresentation (CWE-451) in repository configuration files, malicious repos can trick AI agents into writing to sensitive files (such as SSH keys) without the user realizing it.

Why it matters: As AI agents gain autonomous filesystem read/write access, traditional human-in-the-loop validation prompts become vulnerable to social engineering via UI spoofing, requiring sandboxed execution environments.
